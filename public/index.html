Family First

Family first is a web application designed to assist a group of caregivers with communication of needs for a sick loved one.

The task view is for letting the group know what needs to be done day to day. Tasks are automatically removed after 24hrs to decrease clutter. Adding a task is a single input that automatically tracks who created the task, and who completed the task. There is a comments section to add details later, or edit the info in the task. The medication view is for tracking when medications have been given, and when it is time to administer more medication.
