Family First

Family First combines a simple UI with easy automation and encryption to help keep track of all the tasks associated with caring for a loved one.

There are two views: Tasks and Medications.

The Tasks view allows users to add tasks throughout the day as needed. Tasks are automatically removed after 24hrs to reduce clutter.

The Medication view allows users to input all medications needed. There is an easy to use timer to track when to give medications.

Grid Grinder

Grid Grinder is my introduction to MVC and Backbone. This project made learning Angular much easier.

The challenge was to make a grid based game that has a player that moves around a grid, and loses energy each time it moves.

After succeding in movement with energy loss, I added a energy boost and an energy decreasing bad guy. I intend to keep working on this game for practice and fun. Some ideas are: a score boost/decrease and a constantly moving bad guy.

Apitunes

Apitunes was a weekend group project using angular on the front end and many to one relationships in java on the back end.

We wanted to create a place for artists to upload their songs for the community to listen to them and up/down vote on the songs.

It is designed so that only artists can upload songs, but anyone can vote for songs.

And Friend...

And Friend was an assignment to use 'vanilla' Js to make http requests and manipulate the DOM using the Random User API.

The idea is that a user can create the illusion of having tons of friends by adding or not adding "friends" from a list of random people that is added at a set interval.

Planned additions are a random hobby genereator that will assign each user a random hobby out of a set of 10 hobbies.
